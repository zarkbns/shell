Transaction security. Native to your wallet.

Shell is a modular SDK that intercepts Solana transactions before they reach the blockchain — built to live inside Phantom, Solflare, and Backpack, not alongside them.

How Shell works

Pipeline Intercept

Plugs into the wallet's pre-flight handler. Every transaction is checked before the user sees a confirmation screen.

Away Mode

Sets a local flag that blocks all background signing requests the moment the user steps away.

Mimic Detection

Catches addresses engineered to look like known contacts using heuristic and visual normalization checks.

On-chain Verify

Queries Solana mainnet RPC directly to flag zero-history accounts and program destinations.

Get started

Install 
npm install @shellxyz/sdk
Initialize 
import ShellSDK from '@shellxyz/sdk'

ShellSDK.init({
  contactList: ['your-verified-addresses'],
  solThreshold: 10
})
Intercept transactions 
const result = await ShellSDK.analyzeTransaction({
  address: destinationAddress,
  transferAmount: amount,
  transactionType: 'Direct Transfer'
})

if (result.blocked) rejectTransaction(result.reason)
if (result.requiresPIN) triggerPINChallenge()
